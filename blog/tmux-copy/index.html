Errors validating certificates when running inside a minimal Docker image?

Problem: Automatic copy to clipboard doesn’t work

  • When ssh’d to a remote server
  • Running tmux
  • From a Ghostty terminal window

But does work:

  • When not running tmux, everything else the same.

Solution: On the remote server, update `~/.tmux.conf and add the line:

set -g allow-passthrough on

Certificates in Docker Containers

    Jason Short
  • 26th Jan 2026
  • •1 min read

Errors validating certificates when running inside a minimal Docker image?

If you’re using a minimal image, you might need to add ca-certificates:

FROM alpine AS certs
RUN apk add --no-cache ca-certificates

FROM scratch
COPY --from=certs /etc/ssl/certs/ca-certificates.crt /etc/ssl/certs/
COPY my-go-binary /
CMD ["/my-go-binary"]
